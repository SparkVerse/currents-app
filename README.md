# Currents App 

A currency conversion application built with Next.js and TypeScript. It allows users to convert between multiple currencies using real-time exchange rate data through a clean and responsive interface. 

## Features 

* Real-time currency conversion 
* Multiple currency support 
* Live exchange rate updates 
* Currency selection and swapping 
* Responsive design 
* State management with Zustand 

## Tech Stack 

* Next.js 
* React 
* TypeScript 
* Tailwind CSS 
* Zustand 
* React Query
* ExchangeRate API 
* Git 
* GitHub 

## Overview 

The application enables users to select a base currency and a target currency, enter an amount, and receive a converted value using current exchange rates. State management is handled with Zustand to provide a smooth and efficient user experience. 

## Installation 

### Clone the Repository

bash
git clone <>

### Install Dependencies

bash
npm install

### Run the Development Server

bash
npm run dev

Open http://localhost:3000 in your browser to view the application.
